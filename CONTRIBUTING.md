# Contributing to AI & Blockchain Starter Kits

Thank you for your interest in contributing! This guide will help you get started.

## How to Contribute

### Adding New Projects

1. **Fork the repository**
2. **Create a new branch**
   ```bash
   git checkout -b add/new-project
   ```
3. **Add your project to README.md** following the existing format:
   - Description
   - Why build this
   - Tech Stack
   - Getting Started (code snippets)
   - Key Features
   - Resources

4. **Test your changes** (if applicable)
5. **Submit a PR**

### Project Template

When adding a new project, include:

```markdown
### Project Name
**Description:** Brief description (1-2 sentences)

**Why build this:**
- Reason 1
- Reason 2

**Tech Stack:**
- Tech 1
- Tech 2

**Getting Started:**
\`\`\`bash
# Installation steps
\`\`\`

**Key Features:**
- Feature 1
- Feature 2

**Resources:**
- [Resource 1](url)
- [Resource 2](url)
```

### Guidelines

- **Be descriptive** - Explain why each project is useful
- **Include working code** - All examples should be copy-paste ready
- **Add resources** - Link to official docs and tutorials
- **Keep it updated** - Technologies change fast!

### Reporting Issues

Found a broken link or outdated info? Open an issue with:
- Description of the problem
- Suggested fix (optional)
- Relevant links

---

**Questions?** Open a discussion or reach out on socials!

Happy building! 🚀
